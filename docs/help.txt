                           1. GETTING STARTED
================================================================================

FIRST RUN:
----------
When you first start Enigma, a Setup Wizard will guide you through:
  1. Naming your AI
  2. Choosing a model size (based on your hardware)
  3. Creating the initial model

STARTING THE GUI:
-----------------
    python run.py --gui

OTHER WAYS TO RUN:
------------------
    python run.py --run     # Command line chat
    python run.py --train   # Train from command line
    python run.py --serve   # Start API server

================================================================================
                         3. TRAINING YOUR AI
================================================================================

STEP 1: Prepare Training Data
-----------------------------
  - Go to Data Editor tab
  - Create or edit a training file
  - Use one of these formats:
  
    Q&A Format:
    Q: What is your name?
    A: I am Enigma.
    
    Conversation Format:
    User: Hello
    Assistant: Hello! How can I help?
    
    Plain Text:
    Any text you want the AI to learn from.

STEP 2: Configure Training
--------------------------
  - Go to Training tab
  - Click "Use for Training" in Data Editor (or select file)
  - Set epochs (more = longer training, better learning)
  - Set batch size (lower if you get memory errors)
  - Set learning rate (leave default unless you know what you're doing)

STEP 3: Start Training
----------------------
  - Click Start Training
  - Watch the progress bar
  - Good training shows decreasing loss:
      Epoch 1: loss = 4.2
      Epoch 5: loss = 2.1
      Epoch 10: loss = 0.8

STEP 4: Test Your AI
--------------------
  - Go back to Chat tab
  - Ask questions from your training data
  - If responses are poor, train more or add more data

                         5. TROUBLESHOOTING
================================================================================

"No module named 'torch'"
-------------------------
Install PyTorch:
    pip install torch torchvision torchaudio

"CUDA out of memory"
--------------------
  - Use smaller batch size (1 or 2)
  - Use smaller model size (tiny or small)
  - Close other programs using GPU

"Model produces garbage"
------------------------
  - Need more training data (aim for 10,000+ lines)
  - Train for more epochs
  - Check data format is correct

"GUI not starting"
------------------
Install PyQt5:
    pip install PyQt5

"Voice not working"
-------------------
Install voice dependencies:
    pip install pyttsx3 SpeechRecognition pyaudio

"Screen capture is black"
-------------------------
On Raspberry Pi (Wayland):
    pip install mss pyscreenshot
Or switch to X11 session

"Camera not working"
--------------------
Install OpenCV:
    pip install opencv-python
Check camera permissions

                         6. WHAT NOT TO DO
================================================================================

‚ùå DON'T train with very small datasets
   - Need at least 1000 lines for basic results
   - Need 10,000+ for good results

‚ùå DON'T use extremely high learning rates
   - Default 0.0001 is safe
   - Higher than 0.01 can break training

‚ùå DON'T delete model folders manually
   - Use Model Manager instead
   - Registry needs to be updated

‚ùå DON'T train during inference
   - Close chat before training
   - Or use separate model instances

‚ùå DON'T run multiple instances on same model
   - Can corrupt model files
   - Use different model names

‚ùå DON'T forget to save training data
   - Editor doesn't auto-save
   - Click üíæ Save before training

‚ùå DON'T expect perfect results immediately
   - AI learning takes time
   - Iterate: train, test, improve data, repeat

‚ùå DON'T use copyrighted text for training
   - Stick to your own content
   - Or use public domain data

‚ùå DON'T share trained models publicly without checking
   - May contain patterns from training data
   - Be mindful of what you trained on
