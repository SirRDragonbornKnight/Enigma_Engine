# Information Directory

This directory contains data files used by Enigma Engine.

## Contents

- **conversations/** - Saved conversation histories
- **protocols/** - Communication protocols (API, game, robot)
- **voice_profiles/** - Voice customization presets (GLaDOS, JARVIS, etc.)

## Voice Profiles

Pre-configured voice personalities:
- `default.json` - Standard neutral voice
- `glados.json` - Cold, sarcastic AI (Portal)
- `jarvis.json` - Formal British assistant
- `hal9000.json` - Calm, deliberate AI
- `wheatley.json` - Nervous, rambling AI
- `robot.json` - Classic robotic voice
- `cheerful.json` - Happy, energetic
- `wise.json` - Slow, thoughtful mentor
